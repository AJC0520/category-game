# Category Game

A web-based guessing game inspired by the mobile game "Fight List".

## About

Players are given a category (like "Sports" or "Fruits") and must guess as many answers as possible. Each correct answer awards points based on how obscure it is:
- Common answers: 1 point
- Uncommon answers: 2 points  
- Rare answers: 3 points

## Tech Stack

- React
- TypeScript
- React Router
- Vite

## Getting Started

```bash
npm install
npm run dev
```
